소개

저는 한라대학교 소프트웨어융합학과 4학년 재학생으로, 열정적인 게임 개발자입니다. Unity와 Unreal Engine 5를 통한 게임 개발과 제작에 강점을 가지고 있으며, 게임 시스템, 성능 최적화에 특히 집중하고 있습니다.

  • Unity (C#, VFX, PUN, Mirror)
  • Unreal Engine 5 (C++, Blueprint, PhysX)

프로젝트

멀티플레이어 게임 개발

Photon for Unity와 Mirror를 사용해 온라인/LAN 멀티플레이어 게임을 개발했습니다. vis2k/Mirror 오픈소스 프로젝트에 기여하며 어셈블리 분리 및 에디터 도구 개발에 참여했습니다.

PhysX 성능 최적화

Unreal Engine 5에서 복잡한 물리 장면의 성능 향상을 연구하고 구현했습니다. PhysX 3.4를 통합하고 테스트/성능 지표 수집을 자동화했습니다.

교육 웹사이트 개발

SvelteKit과 Firebase를 사용해 풀스택 교육 웹사이트를 구축했습니다. Firestore와 Edge Functions로 커뮤니티 백엔드를 처리하고, React.js로 쇼핑몰 관리 솔루션 프론트엔드를 개발했습니다.

자동화 도구

Selenium, pyautogui, OpenCV를 활용해 작업 자동화 및 이미지 인식을 위한 Python 기반 자동화 도구를 개발했습니다.
